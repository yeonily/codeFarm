# JSP-Project
너와 농부싶어 ver.1


## 👩‍💻 프로젝트 소개
똑똑! 청년농부 사이트를 벤치마킹하여 만든 2030 청년 귀농 지원 사이트입니다.

귀농귀촌에 관심이 있거나 원하는 사람들에게 필요한 전문 지식 및 정보들을 안내하고 <br>
서로 간의 커뮤니티와 이용할 수 있는 다양한 프로그램, 아르바이트 정보를 제공하여 귀농귀촌에 대한 접근성 향상 도모

## ⏱️ 개발 기간
- 22.09.23 ~ 22.10.25

## 🧑🏻‍🤝‍🧑🏼 맡은 역할
- 퍼블리싱 : 메인페이지, 커뮤니티
- 백엔드 : 프로그램
